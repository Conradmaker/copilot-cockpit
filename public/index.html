Copilot Agent Harness는 VS Code Copilot Chat 위에서 동작하는 구조적 AI 에이전트 개발 시스템입니다. Retrieval-led reasoning, 체계적 게이트, 역할 분업을 통해 긴 대화에서도 품질이 무너지지 않습니다.

Agent System

역할 기반 에이전트 분업

단일 AI가 모든 것을 처리하지 않습니다. 기획, 설계, 실행, 검토 각 단계를 전문화된 에이전트가 맡아 컨텍스트 오염과 품질 저하를 방지합니다.

Phase 1 · Planning 🧭

Mate

사용자 의도를 approved PRD와 downstream seed로 수렴시키는 플래닝 오너. quality gate와 explicit user alignment를 세운다.

📝 prd.md 🔗 references.md
Phase 2 · Design 🎨

Designer

approved PRD를 기반으로 visual, UX, interaction 결정을 execution 이전 문서로 구체화하는 downstream UI/UX 설계 오너.

🖼 design.md
Phase 2 · Technical 🏗

Architector

architecture, integration, stack choice, technical constraints, NFR mapping을 technical.md로 구체화하는 downstream 기술 설계 오너.

⚙️ technical.md
Phase 4 · Review 🔍

Reviewer

security, frontend, design, performance를 reviewer_role 기반으로 독립 검토하고 board gate로 최종 verdict를 합성.

✅ review verdict
Phase 3 · Execution
⚡

Commander

dependency-aware execution plan과 review strategy를 수립하고 todo 기반 진행을 추적하는 실행 오케스트레이터.

📋 execution-plan.md
🤖

Deep Execution Agent

Commander가 위임한 approved scope 안에서 실제 코드 구현과 verification evidence를 생성하는 delegated coding worker.

💻 implementation 🔬 verification
