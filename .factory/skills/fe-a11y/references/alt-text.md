# 이미지 대체 텍스트

이미지나 아이콘을 사용할 때 스크린 리더 사용자를 위해 적절한 대체 텍스트를 제공하는 방법을 다뤄요.

---

## 대체 텍스트가 필요한 경우

이미지나 아이콘이 **정보를 전달하거나 기능을 나타낼 때**는 반드시 의미 있는 대체 텍스트를 제공해야 해요.

```html
<!-- ❌ 의미 있는 아이콘에 대체 텍스트 누락 -->
<button>
  <img src="search-icon.svg" alt="" />
</button>

<!-- ✅ 의미 있는 아이콘에 대체 텍스트 제공 -->
<button>
  <img src="search-icon.svg" alt="검색" />
</button>
```

---

## 대체 텍스트가 불필요한 경우

### 1. 장식용 이미지

단순히 시각적 효과만을 위한 이미지에는 빈 `alt`(`alt=""`)를 사용하세요.

```html
<!-- ❌ 장식용 이미지에 불필요한 텍스트 -->
<img src="divider.png" alt="구분선" />

<!-- ✅ 장식용 이미지에 빈 alt -->
<img src="divider.png" alt="" />
```

### 2. 이미지 주변에 텍스트 설명이 있을 때

이미 충분한 설명이 있다면, 같은 내용을 반복하지 마세요.

```html
<!-- ❌ 중복된 대체 텍스트 → "신제품 스마트폰 신제품 스마트폰"으로 읽힘 -->
<figure>
  <img src="product.jpg" alt="신제품 스마트폰" />
  <figcaption>신제품 스마트폰</figcaption>
</figure>

<!-- ✅ 캡션이 있으면 이미지 alt 생략 -->
<figure>
  <img src="product.jpg" alt="" />
  <figcaption>신제품 스마트폰</figcaption>
</figure>
```

### 3. 아이콘이 텍스트와 함께 사용될 때

```html
<!-- ❌ 중복 → "삭제 아이콘 삭제"로 읽힘 -->
<button>
  <img src="trash-icon.svg" alt="삭제 아이콘" />
  삭제
</button>

<!-- ✅ 텍스트가 있으니 아이콘 alt 생략 -->
<button>
  <img src="trash-icon.svg" alt="" />
  삭제
</button>
```

## 유형별 대체 텍스트 가이드

### 브랜드 로고

로고에 텍스트가 포함되지 않았다면 `alt`로 브랜드명을 전달해요.

```html
<!-- ❌ "보험료 조회하면, 보험료 조회하면" → 어떤 브랜드인지 구분 불가 -->
<img src="logo-carrot.png" alt="" />
<span>보험료 조회하면</span>

<!-- ✅ "캐롯손해보험 보험료 조회하면" → 브랜드 식별 가능 -->
<img src="logo-carrot.png" alt="캐롯손해보험" />
<span>보험료 조회하면</span>
```

> 로고 안에 이미 텍스트로 브랜드명이 있으면 `alt=""`로 처리해요.

### 콘텐츠 이미지

이미지가 주요 정보를 전달할 때는 이미지 자체를 설명해요.

```html
<!-- ✅ 카드 선택 화면에서 어떤 카드인지 식별 -->
<button><img src="gift-card.png" alt="선물 상자 카드" /></button>
<button><img src="diamond-card.png" alt="다이아몬드 카드" /></button>
```

> 이미지가 단순 시각 보조이거나 장식 요소라면 `alt=""`를 사용해요.

### 상품 이미지

상품 이미지에서 텍스트만으로 상품을 식별할 수 없다면 `alt`에 상품명을 포함해요.

```html
<!-- ❌ "4,700원, 아이스아메리카노 T" → 어떤 브랜드인지 모름 -->
<img src="starbucks.png" alt="" />
<span>4,700원</span><span>아이스아메리카노 T</span>

<!-- ✅ "스타벅스, 4,700원, 아이스아메리카노 T" → 브랜드 포함 -->
<img src="starbucks.png" alt="스타벅스" />
<span>4,700원</span><span>아이스아메리카노 T</span>
```

> 텍스트에 이미 상품명이 포함되어 있으면 `alt=""`를 사용해요.

### 로딩/프로그레스 이미지

```html
<!-- ❌ 어떤 변화가 일어나는지 알 수 없음 -->
<img src="typing-indicator.gif" alt="" />

<!-- ✅ 상태를 명확히 전달 -->
<img src="typing-indicator.gif" alt="상담원이 채팅을 입력중입니다" />
```

### 차트/그래프 이미지

축 이름과 주요 값을 함께 안내해요.

```html
<!-- ❌ 데이터를 알 수 없음 -->
<img src="chart.png" alt="" />

<!-- ✅ 핵심 데이터를 전달 -->
<img src="chart.png" alt="원리금균등 163,244원, 원금균등 162,500원, 만기일시 300,000원" />
```

### 프로그레스 바

단계가 중요한 경우 현재 단계를 알려요.

```html
<!-- ❌ "자산 연결 중, 자산 연결 완료" → 몇 단계인지 모름 -->
<img src="progress-step1.png" alt="" />

<!-- ✅ 현재 단계를 명확히 전달 -->
<img src="progress-step1.png" alt="자산 연결 1단계 완료" />
```

## 효과적인 대체 텍스트 작성 원칙

### 1. 명확하고 구체적으로

| 구분 | 예시 | 설명 |
|------|------|------|
| ❌ | "아이콘" | 아이콘의 용도/기능 정보 없음 |
| ✅ | "검색" | 아이콘의 의미를 명확히 전달 |
| ❌ | "그래프" | 무엇을 나타내는지 알 수 없음 |
| ✅ | "2023년 매출 그래프" | 구체적인 데이터 설명 |

### 2. 불필요한 단어 제거

스크린 리더는 요소의 역할을 이미 알고 있으므로 "아이콘", "버튼" 같은 표현은 생략해요.

| 구분 | 예시 | 설명 |
|------|------|------|
| ✅ | "검색" | 간결하고 명확 |
| ❌ | "검색 아이콘" | "아이콘"이 불필요한 중복 |
| ✅ | "닫기" | 간결하고 명확 |
| ❌ | "닫기 버튼" | "버튼"이 불필요한 중복 |

### 3. 이미지의 목적과 맥락 고려

같은 이미지라도 사용 맥락에 따라 다른 대체 텍스트가 필요해요.

```html
<!-- ❌ 시각적 설명만 → 기능을 전달하지 않음 -->
<img src="arrow.png" alt="화살표" />
<img src="arrow.png" alt="화살표" />

<!-- ✅ 맥락에 맞는 기능 설명 -->
<img src="arrow.png" alt="이전 페이지로 이동" />
<img src="arrow.png" alt="다음 페이지로 이동" />
```

## 빠른 체크리스트

- [ ] 정보를 전달하는 이미지에 의미 있는 `alt` 제공
- [ ] 장식용 이미지에 빈 `alt=""` 설정
- [ ] 텍스트와 함께 있는 아이콘에 빈 `alt=""` (중복 방지)
- [ ] 대체 텍스트에 "아이콘", "버튼" 등 불필요한 단어 제외
- [ ] 같은 이미지라도 맥락에 맞게 대체 텍스트 작성
- [ ] 차트/그래프에 핵심 데이터 포함
- [ ] 로딩/프로그레스에 현재 상태 설명
